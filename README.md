## Возможности

- **Генерация кодов:** Поддержка TOTP-кодов по [RFC 6238](https://www.rfc-editor.org/rfc/rfc6238) и HOTP-кодов по [RFC 4226](https://www.rfc-editor.org/rfc/rfc4226).
- **Гибкие параметры:** Поддержка алгоритмов SHA-1, SHA-256 и SHA-512, кодов из 6 или 8 цифр, настраиваемого периода и счетчика.
- **Безопасность и PIN-код:** Защита хранилища 4-значным PIN-кодом. При включении блокировки мастер-ключ шифруется с помощью производного ключа **PBKDF2** (100,000 итераций, соль, SHA-256).
- **Мои категории:** Группировка аккаунтов по категориям. Поддерживаются как встроенные категории (*Личные, Работа, Финансы, Соцсети, Другое*), так и создание/удаление **собственных пользовательских категорий** прямо из меню настроек.
- **Управление порядком (Drag & Drop):** Интуитивная сортировка аккаунтов перетаскиванием мыши/тачем или через специальный клавиатурный режим перемещения (кнопки управления).
- **Импорт из QR-кодов:**
  - из графических файлов (PNG, JPG, WebP);
  - из изображений в буфере обмена (вставка по клику или `Ctrl+V`);
  - сканирование прямо с видимой области экрана текущей вкладки.
- **Поддержка переноса Google Authenticator:** Импорт нескольких TOTP/HOTP аккаунтов за раз из одного QR-кода экспорта Google.
- **Резервное копирование:** Экспорт и импорт зашифрованных резервных копий в формате JSON для надежного переноса данных.
- **Двуязычный интерфейс:** Полная поддержка локализации и перевода (Русский / Английский).
- **Кастомизация UI:** Выбор темы (Темная / Светлая) и пяти акцентных цветов с автокоррекцией контрастности под светлое оформление.

## Установка

Сборка и установка зависимостей не требуются.

1. Склонируйте репозиторий:

   ```bash
   git clone https://github.com/Moii-gh/OnePass-Auth.git
   ```

2. Откройте страницу расширений в браузере:
   - Chrome: `chrome://extensions`
   - Edge: `edge://extensions`
   - Brave: `brave://extensions`

3. Включите **Режим разработчика** (Developer mode).
4. Нажмите **Загрузить распакованное расширение** (Load unpacked).
5. Выберите папку проекта `OnePass-Auth`.
6. Закрепите OnePass Auth на панели браузера.

## Использование

### Добавление вручную
1. Откройте расширение.
2. Нажмите кнопку `+` в правом верхнем углу.
3. Укажите название сервиса, логин, Base32-секрет и выберите категорию.
4. Нажмите кнопку **Добавить аккаунт** (Add account).

### Создание собственных категорий
1. Откройте ручную форму добавления или редактирования аккаунта (кнопка `+` или пункт контекстного меню).
2. В выпадающем списке выбора категории выберите пункт **+ Создать категорию...**.
3. Введите название новой категории и нажмите кнопку **Сохранить**. Новая категория будет мгновенно создана, выбрана для текущего аккаунта и появится на главном экране в виде чипса фильтрации.

### Настройки расширения
Нажмите на иконку шестеренки ⚙️ в заголовке для перехода в настройки:
- **Язык приложения (App Language):** Смена языка интерфейса (Системный по умолчанию / Русский / English) с мгновенным переключением в реальном времени без перезапуска расширения.
- **Тема оформления (App Theme) и Акцентный цвет:** Переключение цветовых тем и оформлений.
- **Режим приватности:** Размытие кодов авторизации до наведения курсора мыши.
- **Блокировка PIN-кодом:** Установка 4-значного PIN-кода для надежной защиты секретов (шифрование PBKDF2).
- **Резервное копирование:** Импорт и экспорт локальных бэкапов в JSON.

### Импорт из QR-кода
Нажмите кнопку QR в заголовке расширения и выберите подходящий способ:
- **Scan Screen** — сделать снимок экрана текущей вкладки и выделить прямоугольную область с QR-кодом;
- **Paste Image** или `Ctrl+V` — вставить картинку QR-кода из буфера обмена;
- Перетащите файл в область загрузки или выберите его вручную.

Перед сохранением расширение покажет список всех распознанных аккаунтов для подтверждения.

### Управление аккаунтами
- Левый клик по карточке копирует текущий код.
- Правый клик открывает контекстное меню: копирование кода, ручное редактирование данных, перемещение по списку (Drag & Drop) или удаление.
- Круговой таймер справа на карточке показывает оставшееся время действия кода.

## Хранение и безопасность

Аккаунты хранятся в защищенном хранилище `chrome.storage.sync` браузера (синхронизируется между вашими устройствами с одним профилем браузера).

Секреты записываются в зашифрованном виде с использованием алгоритма AES-256-GCM. 
- По умолчанию расширение генерирует случайный мастер-ключ.
- Для максимальной защиты включите **Блокировку PIN-кодом** в настройках. При этом мастер-ключ будет надежно зашифрован на основе вашего 4-значного PIN-кода с использованием ресурсоемкой функции хэширования **PBKDF2** (100,000 раундов, случайная соль). Без ввода верного PIN-кода расшифровать аккаунты невозможно.

## Разрешения

| Разрешение | Назначение |
|---|---|
| `storage` | Хранение настроек, категорий и зашифрованных аккаунтов в профиле браузера |
| `activeTab` | Получение снимка экрана текущей страницы по запросу пользователя для сканирования QR |
| `<all_urls>` | Доступ, необходимый для выполнения скрипта выделения области экрана |

## Структура проекта

```text
OnePass-Auth/
├── _locales/       # Локализация расширения (ru, en)
├── icons/          # Иконки приложения
├── manifest.json   # Манифест и разрешения Chrome Extension
├── popup.html      # Разметка главного окна
├── popup.css       # Стили интерфейса (адаптивная сетка, темы)
├── popup.js        # Главный контроллер приложения (ES Module)
├── ui.js           # Управление темами, акцентами и прокруткой
├── drag-drop.js    # Логика сортировки и перетаскивания карточек
├── backup.js       # Логика экспорта/импорта JSON бэкапов
├── qr-scanner.js   # Захват экрана, чтение буфера и загрузка QR-файлов
├── i18n.js         # Перевод интерфейса и всплывающих подсказок
├── crypto.js       # Шифрование данных (AES-GCM-256 + PBKDF2)
├── storage.js      # CRUD операции для аккаунтов и категорий в Chrome Storage
├── totp.js         # Математический расчет OTP (RFC 6238 / RFC 4226)
├── qr.js           # Разбор otpauth URL-ссылок
├── crop.html       # Окно разметки выделения экрана
├── crop.js         # Скрипт выделения области на Canvas
└── jsQR.js         # Внешняя библиотека считывания QR-кодов
```

## Разработка

Для проверки корректности синтаксиса JavaScript модулей можно запустить:

```bash
node --check popup.js ui.js storage.js crypto.js i18n.js backup.js qr-scanner.js drag-drop.js totp.js qr.js crop.js
```
