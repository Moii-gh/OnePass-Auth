## Возможности

- Генерация TOTP-кодов по [RFC 6238](https://www.rfc-editor.org/rfc/rfc6238) и HOTP-кодов по [RFC 4226](https://www.rfc-editor.org/rfc/rfc4226).
- Поддержка SHA-1, SHA-256 и SHA-512, кодов из 6 или 8 цифр, настраиваемого периода и счетчика.
- Ручное добавление аккаунта по Base32-секрету.
- Импорт стандартных `otpauth://totp/` и `otpauth://hotp/` QR-кодов.
- Импорт нескольких TOTP/HOTP аккаунтов из QR-кода переноса Google Authenticator.
- Сканирование QR-кода:
  - из файла PNG, JPG или WebP;
  - из изображения в буфере обмена;
  - прямо с видимой области текущей вкладки.
- Копирование кода одним нажатием.
- Локальное хранение аккаунтов в профиле браузера.
- Шифрование секретов с помощью AES-256-GCM через Web Crypto API.

## Установка

Сборка и установка зависимостей не требуются.

1. Склонируйте репозиторий:

   ```bash
   git clone https://github.com/Moii-gh/OnePass-Auth.git
   ```

2. Откройте страницу расширений:

   - Chrome: `chrome://extensions`
   - Edge: `edge://extensions`
   - Brave: `brave://extensions`

3. Включите **Режим разработчика**.
4. Нажмите **Загрузить распакованное расширение**.
5. Выберите папку `OnePass-Auth`.
6. Закрепите OnePass Auth на панели браузера.

## Использование

### Добавление вручную

1. Откройте расширение.
2. Нажмите кнопку `+`.
3. Укажите сервис, логин и Base32-секрет.
4. Нажмите **Add account**.

### Импорт из QR-кода

Нажмите кнопку QR в заголовке расширения и выберите подходящий способ:

- **Scan Screen** — сделать снимок текущей вкладки и выделить область с QR-кодом;
- **Paste Image** или `Ctrl+V` — вставить изображение из буфера обмена;
- область загрузки — выбрать файл с QR-кодом.

Перед сохранением OnePass Auth покажет распознанные аккаунты для подтверждения.

### Управление аккаунтами

- Левый клик по карточке копирует текущий код.
- Правый клик открывает меню копирования и удаления.
- Таймер справа показывает, сколько секунд осталось до обновления кода.

## Хранение и безопасность

Аккаунты хранятся в `chrome.storage.local`. Секреты записываются в зашифрованном виде с использованием AES-256-GCM.

Важно понимать ограничения текущей реализации:

- ключ шифрования хранится в том же локальном хранилище расширения;
- мастер-пароль и блокировка приложения не реализованы;
- синхронизация, экспорт и резервное копирование отсутствуют;
- удаление расширения или очистка его данных может привести к потере аккаунтов.

Такое шифрование скрывает секреты от простого просмотра данных, но не заменяет специализированное защищённое хранилище при компрометации профиля браузера или устройства.

## Разрешения

| Разрешение | Назначение |
|---|---|
| `storage` | Хранение аккаунтов, ключа шифрования и временного снимка экрана |
| `activeTab` | Работа с активной вкладкой по действию пользователя |
| `<all_urls>` | Доступ, заявленный для сканирования QR-кодов с веб-страниц |

Расширение не содержит серверной синхронизации, аналитики или собственного API.

## Технологии

- Chrome Extensions Manifest V3
- Vanilla JavaScript
- Web Crypto API
- Chrome Storage API
- Canvas API
- [jsQR](https://github.com/cozmo/jsQR)
- HTML5 и CSS3

## Структура проекта

```text
OnePass-Auth/
├── icons/          # Иконки расширения
├── manifest.json   # Манифест и разрешения
├── popup.html      # Интерфейс всплывающего окна
├── popup.css       # Стили интерфейса
├── popup.js        # Управление интерфейсом и аккаунтами
├── totp.js         # Генерация TOTP-кодов
├── crypto.js       # Шифрование и расшифровка секретов
├── storage.js      # Работа с chrome.storage.local
├── qr.js           # Разбор QR и otpauth-ссылок
├── crop.html       # Экран выделения QR-кода
├── crop.js         # Сканирование выделенной области
└── jsQR.js         # Библиотека распознавания QR-кодов
```

## Разработка

После изменения файлов откройте страницу расширений и нажмите кнопку обновления на карточке OnePass Auth. Отдельный dev-сервер или сборщик не используются.

Для быстрой проверки JavaScript-файлов:

```bash
node --check popup.js
node --check totp.js
node --check qr.js
node --check storage.js
node --check crypto.js
node --check crop.js
```
